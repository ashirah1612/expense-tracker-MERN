# MERN EXPENSE TRACKER

A full-stack **Expense Tracker Web App** built using the MERN stack (MongoDB, Express, React, Node.js).

---

## **FEATURES**

- User Signup & Login  
- Add Expenses  
- Edit & Delete Expenses  
- View Expense List  
- Separate user data  

---

## **TECH STACK**

- Frontend: React.js  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- API: REST APIs  
- Styling: CSS  

---

## **PROJECT STRUCTURE**

```
backend/
src/
public/
```

---

## **HOW TO RUN LOCALLY**

1. Install dependencies  
   ```
   npm install
   ```

2. Run frontend  
   ```
   npm start
   ```

3. Run backend  
   ```
   cd backend
   node loginproj.js
   ```
