#  MERN Expense Tracker

A full-stack **Expense Tracker Web App** built using the MERN stack (MongoDB, Express, React, Node.js).

## Features

* User Signup & Login
* Add Expenses
* Edit & Delete Expenses
* View Expense List
* Separate user data 


##  Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* API: REST APIs
* Styling: CSS

## How to Run Locally

 1. Install dependencies

 2. Run frontend
      npm start
 3. Run backend

      node loginproj.js
